Introduction to JavaScript

JavaScript is a versatile programming language used primarily in web development.

It can run in browsers and on servers (Node.js).

JavaScript allows dynamic content, interactive UI, and manipulation of the DOM.

It supports functional, object-oriented, and event-driven programming.

Example of a simple JavaScript statement:

console.log('Hello World');
Variables and Data Types

Variables are used to store data in memory.

JavaScript has dynamic typing, meaning variables can hold any data type.

  • String
  • Number
  • Boolean
  • Object
  • Undefined

Example:

let name = "Alice"; const PI = 3.1416;
Functions

Functions are blocks of code designed to perform specific tasks.

They can take parameters and return values.

  • Function Declaration
  • Function Expression
  • Arrow Functions

Example:

function greet(name) { return `Hello ${name}`; }
DOM Manipulation

The Document Object Model (DOM) represents the HTML structure.

JavaScript can manipulate elements, attributes, and content.

  • getElementById()
  • querySelector()
  • createElement()

Example:

document.getElementById('main-doc');
Events

Events allow interaction between users and the page.

Examples include clicks, mouseovers, and keyboard input.

  • click
  • mouseover
  • keydown

Example:

element.addEventListener('click', () => alert('Clicked!'));
